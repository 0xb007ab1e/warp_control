## Common Shortcuts
| Shortcut          | Action                          |
|-------------------|---------------------------------|
| Ctrl+Shift+T      | Open new tab                   |
| Ctrl+Shift+D      | Split pane horizontally        |
| Ctrl+Shift+H      | Split pane vertically          |
| Ctrl+Shift+C      | Copy                           |
| Ctrl+Shift+V      | Paste                          |

## Dependencies
1. Bash (version 4.0 or higher)
2. Warp Terminal
3. cron (for scheduled tasks)
